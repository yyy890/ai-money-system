## 🚀 5分钟快速启动

### 前置条件
- Mac/Linux/Windows (WSL)
- 能访问Google/OpenAI的代理
- Telegram账号（接收推送）

### 3步部署

**Step 1: 安装Hermes Agent**
```bash
pip install hermes-agent
```

**Step 2: 配置API密钥**
编辑 `~/.hermes/config.yaml`，填入：
- DeepSeek API Key（或其他LLM）
- Telegram Bot Token
- 代理地址

**Step 3: 启动机器人**
```bash
# 复制脚本到Hermes目录
cp scripts/*.py ~/.hermes/scripts/

# 创建定时任务（在Hermes中运行）
cronjob create --schedule "every 2h" --prompt "运行内容套利脚本"
```

**完成！** 系统会自动运行，所有内容推送到你的Telegram。
